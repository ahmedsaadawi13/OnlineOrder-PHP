üõí Create Order

Place a new order with automatic calculations

üì§ Request

POST /api/v1/orders Authorization: Bearer {access_token} Content-Type: application/json { "customer_id": 1, "branch_id": 1, "order_type": "delivery", "delivery_address_id": 1, "items": [ { "item_id": 1, "quantity": 2, "modifiers": [ { "modifier_option_id": 2, "price_adjustment": 3.00 } ] } ] }

‚úÖ Complex Validation Rules

  • branch_id: required, integer, exists in branches table
  • order_type: required, must be one of: dine_in, takeout, delivery
  • items: required, must be array (at least 1 item)
  • items.*.item_id: required, integer
  • items.*.quantity: required, integer, minimum 1
  • special_instructions: max 1000 characters

üí∞ Automatic Calculations

  1. Subtotal: Sum of (item price √ó quantity) for all items
  2. Tax: Applied based on restaurant settings
  3. Delivery Fee: Added for delivery orders (distance-based)
  4. Coupon Discount: Applied if valid coupon provided
  5. Total: Subtotal + Tax + Delivery Fee - Discount
